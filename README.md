## EzService 
EzService est un site web qui permer aux utilisateurs d'échanger un service contre un autre service. 
Le site web contient une liste de compétences (skill) que les utilisateurs pourront choisir pour ensuite proposer leurs expertise à ceux qui en ont besoin.

Pour simplifier l'échange, un service dure une journée entière.

### Les visiteurs
Les visiteurs (utilisateurs non connectés) verront la liste des compétences proposés par le site.

### Les utilisateurs connectés
Les utilisateurs connectés pourront choisir une ou plusieurs compétences qu'ils possèdent et qu'ils seront prêt à échanger contre un autre service.
Ils peuvent aussi voir les autres utilisateurs qui ont proposé des compétences ainsi que leurs nom, prénom, adresse email.
Une fois qu'ils auront repéré une compétence désirée à une date qui leurs convient, l'utilisateur pourra alors réserver ce crénaux pour faire l'échange de compétence en question.

## L'application

### Comment lancer l'application.
1- Cloner le dépot en local
2- Ouvrir pycharm/VsCode dans le dossier django-eval
3- Ouvrir le terminal et taper la commande : cd django-eval
4- Ensuite taper la commande : python manage.py runserver
5- Vous aurez un lien qui s'affiche qui menera vers le site, ensuite il faut aller sur (lien généré) + /ezservice

### Pannel admin
Pour accéder au pannel admin, il faut aller sur le (lien généré) + /admin
Il faudra alors se connecter.
